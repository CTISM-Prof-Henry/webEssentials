Box Model

Este parágrafo está testando todas as propriedades do modelo em caixa do CSS

  • O conteúdo fica no centro (o texto);
  • O Padding é o espaço que existe entre o conteúdo e a borda;
    • Segundo o CSS, é de 50 pixels;
  • A borda delimita o conteúdo visualmente;
    • Segundo o CSS, ela tem 15 pixels de espessura;
  • A margem é o espaço que tem entre dois elementos.
    • Segundo o CSS, ela tem 20 pixels. Ou seja, existe um intervalo de 20 pixels entre a borda e esta lista!
